West Atlanta Built Environment Survey

Amber's first major engagment with web-mapping took place during a participatory action mapping project with a community organization based in West Atlanta. This organization, called the Westside Atlanta Land Trust, wanted to take advantage of their non-profit ArcGIS Online account and make a web map that shows the conditions of the housing stock in their neighborhood. As such, I used ArcMap to build a database which could then be populated using ArcGIS collector mobile application. The point symbols on this map are the result of the countless hours of resident performed data collection.

Oh, the Places You'll Go

Amber's first "homemade" web-map is a product of the week 3 module of the NMP MAP 672 course. In order to make this map, she first had to learn a number of basic concepts regarding Javascript. Concepts include understanding values and types, using expression operators, and both creating and calling on variables and arrays. Even before being able to make this simple web-map, she had to learn how to use Github and Brackets to access and edit HTML documents. While there's not much to it, the creation of this map was an exciting moment for Amber in her quest programming ninja, GitHub warrior, and creator of wicked awesome web-maps.

Energy in the US

Amber's most recent "homemade" web-map is a product of the week 9 module of the NMP MAP 672 course. While it, again, may not seem like much more than a bunch of pretty colored circles, in order to make this map she had to pull on on the concepts she had learned in the previous 8 module. This includes looping, conditional statements, functions, objects, methods, and working with GeoJSON format and JavaScript's Leaflet library. Leaflet specific skills include using pointToLayer functions to create circle markers, filtering the javascript file to only show certain attributes of the data, and adding event listeners. Amber still has a long, windy road ahead of her in her quest programming ninja, GitHub warrior, and creator of wicked awesome web-maps, but with the creation of this map, it is undeniable that she is one step closer.
